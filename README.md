# Attractive UI Project

A modern React + Vite starter with polished visuals and Docker deployment.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Docker build

```bash
docker build -t attractive-ui-project .
```

## Docker run

```bash
docker run -p 8080:80 attractive-ui-project
```
